WHAT YOU NEED TO DO:
-------------------
YES, you need Facebook credentials:

1. CREATE FACEBOOK APP
   - Go to https://developers.facebook.com/
   - Create a new app (Business type)
   - Get App ID and App Secret

2. GET PAGE ACCESS TOKEN
   - Use Graph API Explorer: https://developers.facebook.com/tools/explorer/
   - Generate a Long-Lived Page Access Token
   - Get your Facebook Page ID

3. CONFIGURE BACKEND
   - Create backend/.env file (copy from backend/.env.example)
   - Add these variables:
     FACEBOOK_PAGE_ID=your_page_id
     FACEBOOK_ACCESS_TOKEN=your_token
     FACEBOOK_APP_ID=your_app_id
     FACEBOOK_APP_SECRET=your_app_secret

4. CONFIGURE FRONTEND
   - Create front/.env file
   - Add: VITE_API_URL=http://localhost:5000/api

5. START SERVERS
   - Backend: cd backend && npm start
   - Frontend: cd front && npm run dev

6. TEST
   - Visit the community showcase section
   - Should see posts from your Facebook Page!
